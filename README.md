# BeatThreat

BeatThreat es una aplicación social diseñada para los amantes de la música, de los artistas y sensaciones que estos emiten a través de sus canciones. El enfoque que
se le quiere brindar es darle la oportunidad a los usuarios la opción de poder calificar albúmes de sus artistas preferidos, generar toda una experiencia social a
partir de la misma, compartir opiniones y participar en debates dentro de un foro social dinámico.

BeatThreat nace con un objetivo principal y es el de crear una comunidad donde los usuarios puedan:
-Descubrir nuevos álbumes.
-Calificar y reseñar estos mismos.
-Participar en debates generados por la propia comunidad, fomentando la multiculturalidad.
-Interactuar con usuarios que tengan gustos a fines
-Visualizar los rankings basados en las calificaciones brindadas por la comunidad.

Las tecnologías que se van a utilizar para el desarrollo de este proyecto son:
-Plataforma de enfoque: AndroidSO
-Entorno de desarrollo: Android Studio
-Lenguaje: Kotlin
-Control de Versiones: Git & Github

Funcionalidades Principales:
-Registro e inicio de Sesión
-Perfil Personalizado
-Búsqueda de artistas y álbumes
-Sistema de calificación númerico
-Publicación de reseñas
-Foro de integración por álbum o artista
-Ranking de álbumes mejor calificados
